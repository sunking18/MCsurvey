# 问卷星接入操作说明

> 配套报告页地址：https://40bd8150a8db4263a5096466570cfced.app.codebuddy.work/

---

## 一、整体思路

1. 在问卷星上创建一份包含「基础信息 + 34 道题目」的测评问卷。
2. 设置 3 个计分维度：家庭沟通（12 题）、学业焦虑（12 题）、心理韧性（10 题）。
3. 沟通量表中的第 4、8、9 题为反向计分，必须在问卷星里设置反向计分。
4. 用户答完后，问卷星自动计算三个维度总分，并跳转到报告页。
5. 报告页根据 URL 中的三个分数，渲染出温暖横幅、分数卡片、雷达图、解释文案和课程建议。

---

## 二、导入问卷题目

### 2.1 文本导入

1. 登录问卷星后台 → 创建问卷 → 选择「文本导入」。
2. 打开本目录下的 `wenjuanxing_import.txt`，复制全部内容。
3. 粘贴到问卷星的文本导入框中，按提示完成识别。
4. 导入后，需要手动调整每个题目的选项分数：
   - **所有 34 道单选题** 均为 5 点量表，选项分值统一为 **1、2、3、4、5 分**。
   - 第 1 个选项 = 1 分，第 5 个选项 = 5 分。

### 2.2 多子女信息逻辑设置

基础信息中已包含「子女 1 / 子女 2 / 子女 3」的年龄和性别字段。建议在问卷星后台为子女 2、子女 3 的字段设置「显示逻辑」：

| 字段 | 显示条件 |
|---|---|
| 子女 2 年龄、子女 2 性别 | 当「您有几个子女」= 2个 / 3个 / 4个及以上 |
| 子女 3 年龄、子女 3 性别 | 当「您有几个子女」= 3个 / 4个及以上 |

> 问卷星的文本导入格式暂不支持直接设置逻辑跳转，导入后请在「逻辑」设置中手动添加。

## 三、三个子问卷的指导语设置

导入模板中已将 34 题分为三部分，每部分包含一段描述说明。导入后，请确保每部分开头保留一段「描述说明」类型的题目（无选项），内容如下：

- **第一部分：家庭沟通**  
  「本部分包含12个题项，请仔细阅读以下每一项目，并根据您的实际情况选择一个最符合您的评分，表明这句话对您而言的正确程度。题目没有对错之分，请根据您的真实感受作答。」

- **第二部分：学业焦虑**  
  「本部分包含12个题项，请仔细阅读以下每一项目，并根据您的实际情况选择一个最符合您的评分，表明这句话对您而言的正确程度。题目没有对错之分，请根据您的真实感受作答。」

- **第三部分：心理韧性**  
  「本部分包含10个题项，请仔细阅读以下每一项目，并根据您的实际情况选择一个最符合您的评分，表明这句话对您而言的正确程度。题目没有对错之分，请根据您的真实感受作答。」

> 注意不要显示「家庭沟通/学业焦虑/心理韧性」等量表名称，避免暴露测量目的。导入模板中已按「第一部分/第二部分/第三部分」呈现，仅作为后台识别使用，职工填写时不会直接看到量表名。

## 四、设置三维度计分

### 4.1 创建测评维度

在问卷星「测评」功能中，创建 3 个维度：

| 维度名称 | 题目范围 | 计分方式 |
|---|---|---|
| 家庭沟通 | 第 1-12 题（对应 34 题中的前 12 题） | 求和 |
| 学业焦虑 | 第 13-24 题 | 求和 |
| 心理韧性 | 第 25-34 题 | 求和 |

### 4.2 设置反向计分（关键）

家庭沟通量表中的第 4、8、9 题（即整套问卷的第 4、8、9 题）为反向计分题。在问卷星中需要单独设置这些题为反向计分：

- 原始选项分值：1、2、3、4、5
- 反向计分后：5、4、3、2、1

> 如果问卷星没有直接提供反向计分功能，也可以在报告页计算时默认「您已处理了反向计分」，即问卷星传出的总分已经是反向计分后的结果。但最稳妥的做法是在问卷星中设置反向计分，确保总分正确。

具体反向题：

- 第 4 题：和孩子讨论事情时，我常常坚持自己说了算。
- 第 8 题：孩子不认同我的观点时，我通常会很生气。
- 第 9 题：孩子做得不好时，我会先批评，而不是先了解原因和想法。

## 五、结束页跳转配置

### 5.1 打开结束页设置

问卷编辑完成后 → 进入「设置」→「提交后」→ 选择「跳转到其他网页」。

### 5.2 填写跳转 URL

跳转地址模板：

```
https://40bd8150a8db4263a5096466570cfced.app.codebuddy.work/?c=家庭沟通得分&a=学业焦虑得分&r=心理韧性得分
```

在问卷星中，需要用变量替换三个分数。不同版本问卷星的变量写法可能不同，常见形式如下：

```
https://40bd8150a8db4263a5096466570cfced.app.codebuddy.work/?c={{维度1得分}}&a={{维度2得分}}&r={{维度3得分}}
```

或：

```
https://40bd8150a8db4263a5096466570cfced.app.codebuddy.work/?c={{so1}}&a={{so2}}&r={{so3}}
```

> **具体变量名以您问卷星后台「测评变量」显示为准。** 通常维度得分变量可以在结束页插入「测评结果」变量时看到。如果找不到，建议先发布测试版，填一份答案后查看 URL 中实际传递的参数。

### 5.3 参数含义

| 参数 | 含义 | 量表满分 |
|---|---|---|
| `c` | 家庭沟通总分 | 12-60 |
| `a` | 学业焦虑总分 | 12-60 |
| `r` | 心理韧性总分 | 10-50 |

## 七、报告页说明

职工提交问卷后，会自动跳转到报告页，看到：

1. **温暖提示横幅**：根据三个维度中最严重等级显示不同文案，整体积极正向、温暖鼓励。
2. **三维度分数卡片**：显示家庭沟通、学业焦虑、心理韧性得分及等级。
3. **雷达图**：直观展示三维度关注指数。
4. **分维度解释**：针对每个维度的具体描述文案。
5. **心理韧性科普**：解释什么是心理韧性及其对儿童成长的意义。
6. **课程建议**：根据较弱项推荐心理团辅课程。
7. **温暖结束语**：鼓励家长持续陪伴。

> 注意：问卷星跳转的报告页为「简版报告」，仅含总分信息，**不显示逐题维度得分条**。完整报告（含逐题维度条）只在直接访问 `index.html` 完整填写时生成。

## 九、常见问题

### Q1：报告页显示 NaN 或空白？

请检查跳转 URL 中的三个参数是否都已正确传递，且为数字。常见原因：
- 变量名写错，导致参数为空。
- 维度得分未正确计算。
- 反向计分未设置，导致维度分数异常。

### Q2：如何收集数据？

当前问卷星方案下，原始数据会保存在问卷星后台。如果需要同步到自建系统，需要：
- 方案 A：问卷星导出 Excel 后人工整理。
- 方案 B：使用问卷星 API 或 Webhook 将数据推送到自建后端（本目录下 `backend/server.js` 可作为参考，但需要部署到有 Node.js 运行环境的服务器）。

### Q3：能否不显示「第一部分/第二部分/第三部分」？

可以。在问卷星中，这部分文字放在「描述说明」题型中，职工看到的是一段温暖的引导语，不会看到「家庭沟通/学业焦虑/心理韧性」等量表名。如果您希望更隐蔽，可以把描述说明里的文字进一步模糊化，例如只写：「请根据最近一个月的实际情况，对以下描述进行评分。」
